# 🎉 Booking Platforma za Igraonice i Rođendaonice

> Centralizirana web aplikacija koja povezuje roditelje, igraonice i dodatne usluge na jednom mjestu.

## Što je ovo?

Platforma za online rezervacije termina u igraonicama i rođendaonicama — roditeljima omogućava pregled, rezervaciju i plaćanje, a igraonicama kompletno upravljanje poslovanjem.

## Za koga je platforma?

| Korisnik | Što dobiva |
|---|---|
| 👨‍👩‍👧 Roditelji | Pregled igraonica, rezervacija, akontacija, recenzije |
| 🏢 Igraonice | Upravljanje kalendarom, rezervacijama, paketima i uplatama |
| 🎭 Animatori & usluge | Marketplace za dodatne usluge |
| 🛡️ Admin | Moderacija, analytics, podrška |

## Tech Stack

```
Frontend   → React + TypeScript + Vite + SCSS + Zustand + TanStack Query
Backend    → Node.js + Express + TypeScript + Prisma
Database   → PostgreSQL
Storage    → Cloudinary / AWS S3
Payments   → Stripe + CorvusPay
Auth       → JWT + Refresh Tokens
```

## MVP Timeline

```
Mjesec 1 → Setup, Auth, Listing, Booking Engine
Mjesec 2 → Payments, Reviews, Dashboard, Deploy
Mjesec 3 → Marketing, Onboarding partnera, Analytics
```
